CSS Examples and Explanations

1. What is CSS?
CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, paper, or in other media.

2. What are CSS Selectors?
CSS Selectors are patterns used to select the elements you want to style. Examples include element selectors, class selectors, and ID selectors.

3. How do you add CSS to a webpage?
CSS can be added to HTML documents in three ways: Inline, Internal, and External. Inline uses the `style` attribute inside HTML elements. Internal uses the `<style>` element inside the `<head>` section. External links to an external CSS file using the `<link>` element.

4. What are CSS Media Queries?
Media queries are used to apply different styles for different devices or screen sizes. They help in making a responsive design that works well on all devices.

            
                @media (max-width: 600px) {
                    .container {
                        flex-direction: column;
                    }
                }

5. What is Flexbox?
Flexbox is a one-dimensional layout method for arranging items in rows or columns. It allows you to distribute space within an item and align items within a container.

                }
